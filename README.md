# Sibol - Web App

> **Growth through social impact.** > *“Sibol” (n.) – Filipino for growth, sprout, or the beginning of a journey.*

Sibol is a minimalist, two-sided marketplace designed to bridge the gap between entry-level developers and startup businesses or NGOs. It provides a "calm" space where early-career talent can build verified proof-of-work while providing essential digital solutions to organizations that need them most.

---

## The Vision
In a digital landscape filled with noise, Sibol focuses on the essentials:
- **For Creators:** Transition from "tutorial hell" to real-world impact with verified project stamps.
- **For Organizations:** Access dedicated talent for introductory websites and digital tools without the corporate friction.
- **The Philosophy:** Minimalist UI, performance-first code, and professional connectivity.

---

## Tech Stack
This project is built with a focus on high-performance "Vanilla" standards and a lightweight development workflow.

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Aceternity UI](https://ui.aceternity.com/)
- **Backend-as-a-Service:** [Supabase](https://supabase.com/) (Auth, Database, RLS)
- **Email Engine:** [Resend API](https://resend.com/)
- **Type Safety:** TypeScript
- **Environment:** Developed using **Sublime Text** for a lightweight, crash-resistant workflow. 

---

## Key Features
- **Dual-Role Interface:** Tailored dashboards for both Developers (Creators) and Organizations (Clients).
- **The Connect Engine:** A seamless bridge for organizations to request connections with developers directly via email.
- **Verified Portfolios:** A minimalist, grid-based display of completed projects with live links.
- **Zero-Waste UI:** High-contrast, monochromatic design optimized for mobile and desktop performance.
